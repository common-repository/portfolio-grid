Portfolio Plugin allows you to display portfolio items in a grid with a interactive filtering system.

== Description ==

Portfolio Plugin will display custom portfolio items into a grid in a responsive wen layout. Using javascript, it will create an interactive filter for portfolio items based on custom filter tags that each project is categorized under. 

== Installation ==

1. Upload `portfolio-grid` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Create a page with the slug `portfolio` for the plugin to display the custom portfolio items
